```

```

## 注解Ent模式

`entsql`包允许使用数据库模式名称注解Ent模式。例如：

```go
// Annotations of the User.
func (User) Annotations() []schema.Annotation {
	return []schema.Annotation{
		entsql.Schema("db3"),
	}
}
```

要在多个Ent模式间共享相同的模式配置，可以使用`ent.Mixin`或定义并嵌入_基础_模式：

<Tabs>
<TabItem value="mixin" label="Mixin schema">

```go title="mixin.go"
// Mixin holds the default configuration for most schemas in this package.
type Mixin struct {
	mixin.Schema
}

// Annotations of the Mixin.
func (Mixin) Annotations() []schema.Annotation {
	return []schema.Annotation{
		entsql.Schema("db1"),
	}
}
```

```go title="user.go"
// User holds the edge schema definition of the User entity.
type User struct {
	ent.Schema
}

// Mixin defines the schemas that mixed into this schema.
func (User) Mixin() []ent.Mixin {
	return []ent.Mixin{
//highlight-next-line
		Mixin{},
	}
}
```

</TabItem>
<TabItem value="base" label="Base schema">

```go title="base.go"
// base holds the default configuration for most schemas in this package.
type base struct {
	ent.Schema
}

// Annotations of the base schema.
func (base) Annotations() []schema.Annotation {
	return []schema.Annotation{
		entsql.Schema("db1"),
	}
}
```

```go title="user.go"
// User holds the edge schema definition of the User entity.
type User struct {
//highlight-next-line
    base
}
```

</TabItem>
</Tabs>

## 生成迁移

要生成迁移，使用`atlas migrate diff`命令。例如：

<Tabs
defaultValue="mysql"
values={[
{label: 'MySQL', value: 'mysql'},
{label: 'MariaDB', value: 'maria'},
{label: 'PostgreSQL', value: 'postgres'},
]}>
<TabItem value="mysql">

```shell
atlas migrate diff \
  --to "ent://ent/schema" \
  --dev-url "docker://mysql/8"
```

</TabItem>
<TabItem value="maria">

```shell
atlas migrate diff \
  --to "ent://ent/schema" \
  --dev-url "docker://maria/8"
```

</TabItem>
<TabItem value="postgres">

```shell
atlas migrate diff \
  --to "ent://ent/schema" \
  --dev-url "docker://postgres/15/dev"
```

</TabItem>
</Tabs>

:::note
默认情况下，`migrate diff`命令生成的SQL语句列表不带缩进。如需生成带缩进的SQL语句，可使用`--format`标志。例如：

```shell
atlas migrate diff \
  --to "ent://ent/schema" \
  --dev-url "docker://postgres/15/dev" \
// highlight-next-line
  --format "{{ sql . \"  \" }}"
```
:::
